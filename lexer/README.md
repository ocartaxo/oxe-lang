# Criando um analisador léxico utilizando o Flex

## Como compilar e executar o analisador léxico

1. Execute o comando abaixo
    ```sh
    make
    ```

2. Execute o lexer passando o arquivo de exemplo

   ```sh
   lexer exemplos/exemplo.bais
   ```
